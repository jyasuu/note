## TOEIC 測驗題型（新版）

### 1. **聽力測驗 Listening Comprehension**（約 45 分鐘，共 100 題）

分成四個部分：

1. **照片描述 (Photographs)**

   * 題數：6 題
   * 聽一段描述，選出與圖片相符的句子。
2. **應答問題 (Question-Response)**

   * 題數：25 題
   * 聽一個問題，從三個回應中選出最適合的。
3. **簡短對話 (Conversations)**

   * 題數：39 題
   * 聽一段對話，回答相關問題（每段對話 3 題左右）。
4. **簡短演說 (Talks)**

   * 題數：30 題
   * 聽一段簡短的演講或廣播，回答相關問題（每段 3 題左右）。

---

### 2. **閱讀測驗 Reading Comprehension**（約 75 分鐘，共 100 題）

分成三個部分：
5\. **填空測驗 (Incomplete Sentences)**

* 題數：30 題
* 考文法、片語、單字，從四個選項中選出最適合的字或詞。

6. **文本填空 (Text Completion)**

   * 題數：16 題
   * 文章中有空格，必須選出正確的選項填入。
   * 常見題材：email、公告、備忘錄、信件。
7. **閱讀理解 (Reading Comprehension)**

   * 題數：54 題
   * 單篇閱讀（短文）、雙篇閱讀、三篇閱讀都有。
   * 題材多為商業、旅遊、辦公室、廣告等日常職場情境。

---

✅ **總計**：200 題，測驗時間約 2 小時（聽力 45 分鐘 + 閱讀 75 分鐘）。

### Part 1. 照片描述 (Photographs)

    // --- Question submit/reset/feedback ---
    let score = 0, answered = 0;
    const scoreEl = document.getElementById('score');
    const answeredEl = document.getElementById('answered');

    function handleSubmit(form){
      const correct = form.getAttribute('data-correct');
      const checked = form.querySelector('input[type="radio"]:checked');
      const feedback = form.querySelector('[data-feedback]');
      if(!checked){
        feedback.textContent = '請先選擇一個選項。';
        feedback.className = 'text-sm text-amber-600';
        return;
      }
      answered++;
      if(checked.value === correct){
        score++;
        feedback.textContent = '✅ 正確！';
        feedback.className = 'text-sm text-emerald-700';
      } else {
        feedback.textContent = `❌ 錯誤。正解為 (${correct}).`;
        feedback.className = 'text-sm text-rose-700';
      }
      // disable options after submit
      form.querySelectorAll('input[type="radio"]').forEach(i => i.disabled = true);
      form.querySelectorAll('[data-submit]').forEach(b => b.disabled = true);
      scoreEl.textContent = score; answeredEl.textContent = answered;
    }

    function handleReset(form){
      form.reset();
      form.querySelectorAll('input[type="radio"]').forEach(i => i.disabled = false);
      form.querySelector('[data-feedback]').textContent = '';
      form.querySelectorAll('[data-submit]').forEach(b => b.disabled = false);
    }

    document.querySelectorAll('form[data-question]').forEach(form => {
      form.querySelector('[data-submit]').addEventListener('click', () => handleSubmit(form));
      form.querySelector('[data-reset]').addEventListener('click', () => handleReset(form));
    });

    // --- Timer ---
    let startAt = null, timerId = null;
    const timerEl = document.getElementById('timer');
    const fmt = (n)=> n.toString().padStart(2,'0');
    function tick(){
      const sec = Math.floor((Date.now() - startAt)/1000);
      const m = Math.floor(sec/60), s = sec%60;
      timerEl.textContent = `${fmt(m)}:${fmt(s)}`;
    }
    document.getElementById('startTimer').addEventListener('click', ()=>{
      if(timerId) return; startAt = Date.now(); timerId = setInterval(tick, 250);
    });
    document.getElementById('stopTimer').addEventListener('click', ()=>{
      clearInterval(timerId); timerId = null;
    });
    document.getElementById('resetTimer').addEventListener('click', ()=>{
      clearInterval(timerId); timerId = null; timerEl.textContent = '00:00';
    });

    // --- Playback Rate (demo) ---
    const rateSel = document.getElementById('playbackRate');
    rateSel.addEventListener('change', ()=>{
      document.querySelectorAll('audio').forEach(a => a.playbackRate = parseFloat(rateSel.value));
    });
